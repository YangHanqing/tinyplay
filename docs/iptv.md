# IPTV

IPTV 就是通过网络看的电视直播。TinyPlay 可以读取 IPTV 播放列表，把频道放到手机上浏览，再让接电视的电脑播放；切换频道、暂停和跳回直播都还是同一套遥控器。

## 怎么使用

1. 打开 TinyPlay 的“设置”，点击“添加内容源”。
2. 选择“IPTV”，输入 M3U 或 M3U8 播放列表地址并保存。
3. 回到“媒体库”，选择频道分类，点击频道开始播放。

播放列表可以去 GitHub 搜索 M3U 或 M3U8，也可以在闲鱼上找现成的播放列表。

## 播放卡顿怎么办

IPTV 播放列表的质量差别很大。频道卡顿、失联、画质不稳定，通常是播放列表、频道源或线路本身的问题，不是 TinyPlay 能解决的。

TinyPlay 负责读取列表和播放频道，不能把质量不好的 IPTV 源变得稳定。
