# 网页控制（中文 Beta）

TinyPlay 可以在接电视的电脑上打开哔哩哔哩、爱奇艺、腾讯视频和优酷，再用手机控制网页。它只在用户自己的电脑上提供网页遥控；用户仍需自行登录，并遵守相应服务的账号、会员、地区和终端使用规则。

## 怎么使用

1. 在手机的“网址库”中选择一个网站。
2. 网站会在电视连接的电脑上打开。
3. 回到手机的“遥控器”，控制播放、快退、快进、倍速和全屏。

网页控制本质上是把电脑上的网页操作，换成更适合沙发的手机遥控器。它能代替鼠标快速点按按钮，也能进入或退出全屏。

这是一个中文 Beta 功能。目前只支持上面的四个网站；网站本身改版时，个别控制偶尔可能需要跟着调整。它不提供去广告、跳过登录、规避会员或终端限制、地区访问绕过、下载、录制或转播；电脑端原本如何获得和使用服务，这里就如何使用。
