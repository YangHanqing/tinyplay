# 直接播放文件

不准备搭建媒体服务器也可以使用 TinyPlay。它可以浏览本地文件夹、已挂载目录、SMB 共享和 WebDAV 共享；连通后，用手机选择起始文件夹，再按目录浏览并播放。先看片，不必先折腾海报墙或插件。

这种方式保留遥控器的暂停、进度、字幕、音轨和倍速控制，但不会生成海报、剧集信息或续播记录。

## macOS

### 播放本机或已挂载目录

在“添加内容源”中选择“本地文件夹”，然后在目录选择器中浏览这台 Mac 上的文件夹。

如果文件放在 NAS 上，也可以先在“访达 → 前往 → 连接服务器”中挂载共享，再把挂载后的目录选作内容源。

### 直接连接 NAS 共享

选择“SMB 共享”，填写共享地址和账号密码。连接成功后，直接在 TinyPlay 中选择需要浏览的共享和文件夹。

## Windows

### 播放本机或已挂载目录

选择“本地文件夹”，再在目录选择器中浏览任意磁盘。也可以先在“此电脑”中映射网络驱动器，再选择映射后的目录。

### 直接连接 NAS 共享

选择“SMB 共享”，填写共享地址和账号密码。连接成功后，在 TinyPlay 中选择共享和起始文件夹，无需先映射驱动器。

::: warning 提示
TinyPlay 运行在接电视的电脑上，因此“本地文件夹”指的是这台电脑能够访问的路径，而不是手机上的文件。
:::
