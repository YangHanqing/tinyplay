# 接收 DLNA 投屏

DLNA 可以理解成一种通用的投屏协议。手机上的播放器找到播放设备后，就可以把视频、音乐或照片投过去。

哔哩哔哩、爱奇艺、腾讯视频、优酷等常见播放器里，通常都能找到“投屏”按钮。

## 用 TinyPlay 接收投屏

Windows 和 Mac 默认不能直接接收 DLNA 投屏。TinyPlay 集成了 DLNA 接收器，安装并运行后，这台接电视的电脑就可以出现在手机播放器的投屏设备列表里。

使用时：

1. 确认手机和运行 TinyPlay 的电脑连接的是同一个局域网。
2. 在手机播放器中点击“投屏”，选择运行 TinyPlay 的电脑。
3. 选择视频，电脑会通过电视播放。

## 倍速控制

DLNA 投屏后，官方 App 不支持直接操作倍速。需要调整播放速度时，可以回到 TinyPlay 手机上的“遥控器”页面，在遥控器里选择倍速。

DLNA 只负责把内容投到另一台设备，不会把弹幕一起带过去，也不能解决有版权保护的视频播放问题。
