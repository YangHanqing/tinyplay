# 搭建媒体库

如果你希望拥有海报墙、剧集选集、搜索和续播体验，先搭建一个媒体服务器，再让 TinyPlay 连接它会更合适。服务器整理内容，TinyPlay 让你用手机选片、由接电视的电脑播放。

## 推荐：Jellyfin

对于中文用户，Jellyfin 是一个直接而稳妥的选择：完全免费开源，支持 Docker，中文资料和社区较多，也无需注册海外账号。群晖、威联通、极空间等主流 NAS 系统通常也提供方便的安装方式。

搭建完成后，在 TinyPlay 的“添加内容源”中选择“Jellyfin”，填入服务器地址与账号即可。

## 先把目录整理好

媒体服务器需要知道影片和剧集放在哪里。建议将电影、电视剧、动画等内容分开存放，并让剧集目录按“剧名 / 季 / 文件”的层级整理。这样服务器更容易匹配海报、剧集信息和字幕。

```text
媒体/
├── 电影/
│   └── 电影名称 (年份).mkv
└── 电视剧/
    └── 剧集名称/
        └── Season 01/
            └── S01E01.mkv
```

## 不必一次配置完整

先让一个文件夹被媒体服务器正确识别，再用 TinyPlay 连接并播放一部影片，就足够验证整套流程。封面、元数据、远程访问等细节可以在之后按需要逐步补齐。
