# 注意事项

## 屏保机制

TinyPlay 的屏保很简单，主要目的是避免电视长时间停在同一幅画面上，减少烧屏风险。

当视频暂停超过五分钟后，屏保会出现。它会把当前影片的海报作为背景并压暗，同时显示时间、日期和影片名称。背景图片大约每分钟更换一次。

继续播放、按下遥控器上的控制按钮，或停止播放后，屏保就会消失，回到正常画面。

它不是一个复杂的系统屏保，也不是为了省电或保护隐私设计的。目前它只做最重要的一件事：避免电视长时间显示同一幅静止画面。

## 安全提醒

TinyPlay 目前是按家庭局域网使用来设计的。应用已经做了一些基本防护，但网页控制、账号权限和公网访问方面还没有做到足够完善。

请不要把 TinyPlay 运行时给手机打开的控制页面暴露到公网，也不要通过端口映射、反向代理或网络隧道直接开放给互联网。

这里说的是 TinyPlay 在电脑上运行的控制页面，不是 GitHub Pages 上的产品介绍网站。日常使用时，让手机和电脑保持在自己信任的家庭局域网中即可。

## 网页服务使用边界

网页控制只在用户自己的 Windows 或 macOS 电脑上操作已打开的网站。请自行登录，并遵守网站关于账号、会员、地区和终端使用的规则。

TinyPlay 不提供 VPN、代理或地区访问绕过，也不提供去广告、会员或终端限制规避、下载、录制、内容转播或流地址导出功能。服务提供方调整网页、会员或终端规则后，以其规则为准。
