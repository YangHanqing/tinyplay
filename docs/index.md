# 闲置 PC，接管电视 🏠

把闲置的 Windows PC、NUC 或 Mac mini 接上电视，再运行 TinyPlay，手机就是它的遥控器。选片、搜索、拖进度、切字幕，都在一套现代的手机界面里完成；电脑端已有的内容和已登录网页服务，可按各服务规则在大屏上使用。

TinyPlay 不是另一台需要学习的电视盒子，也不要求你先搭完整媒体库、研究一堆插件。你可以先连 Jellyfin，也可以直接打开 NAS 文件夹、导入 IPTV；从已有的内容开始就行。

## TinyPlay 可以做什么 📺

TinyPlay 主要用于以下几种场景：

- 🎬 看电影和其他影片
- 📡 看 IPTV
- 📲 接收手机或其他设备的投屏
- 🌐 在电脑端打开哔哩哔哩、爱奇艺、腾讯视频、优酷等网站，再用手机控制（中文 Beta）

## 选择你的起点 🧭

### 🔥 我是发烧友：我已经在使用媒体服务器或 NAS

如果你已经在使用 Emby、Jellyfin 或 Plex 等媒体服务器，或者已经有 NAS，直接阅读[连接媒体服务器](/media-server)即可。

### 🌱 我是小白：我只有电脑上的影音文件夹

如果你只是把下载好的电影、电视剧放在电脑或 NAS 的某个文件夹里，也可以直接播放，不需要先搭建海报墙。TinyPlay 可以浏览这个文件夹，你选择影片后，再用手机控制播放。

如果你希望看到电影海报、分类、剧集信息和续播记录，就可以再搭建 Jellyfin 这类媒体库。媒体库软件负责扫描和整理影片，TinyPlay 则负责让它在手机上浏览、在接电视的电脑上播放。可以从[搭建媒体库](/library)开始。
